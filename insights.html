1. ИИ-АГЕНТЫ КАК СОТРУДНИКИ — 15+ стартапов

Новое: Avoca доказал — та же технология в рынке с ценой проблемы x1000 = x1000 к оценке.


2. ФИЗИЧЕСКИЙ МИР — 6 стартапов, крупнейший раунд $125M

Avoca (HVAC/сантехника, $1B), Cloneable (инспекции, x100 ARR), HVAC-ассистент ($30M), домашний ремонт, medspa.

Самая незатронутая ИИ вертикаль. Пропущенный звонок = $30K потерь. Конкурентов мало.


3. HR-TECH / РЕКРУТИНГ — 7 стартапов

Поиск по 800M профилей ($850M), маркетплейсы рекрутёров ($40M), оценка AI-навыков, прозрачность рынка.

Парадокс: ИИ сокращает найм, но резко поднимает ставки за топ-10%.


4. LEGALTECH — 3 стартапа, все крупные

Договоры с фиксированной ценой ($400M за 286 дней), ИИ-маркетинг для юрфирм ($22M), Manifest OS — франшиза ИИ-нативных юрфирм ($750M за 18 месяцев).

LegalTech взрывается: три разных угла атаки, все крупные раунды.


5. ДАННЫЕ КАК ИНФРАСТРУКТУРА — 4 стартапа

Redpine (лицензированные данные для агентов, $8M), голосовые данные ($25M+), база отелей, Glimpse (финансовые данные поставщиков).

Тренд: ИИ-агентам нужны данные которых нет в публичном интернете — рождается новый слой дата-брокеров.


6. СОЦИАЛЬНЫЕ СЕТИ ДЛЯ ИИ-АГЕНТОВ — 3 стартапа

AgentDiscuss (Product Hunt для агентов), Marx (соцсеть агентов-инвесторов), Humwork (маркетплейс агент→человек).

Новый тренд: агенты учатся друг у друга без участия человека.


7. ПРОГРАММЫ ЛОЯЛЬНОСТИ → ИСТОЧНИК ДОХОДА — 3 стартапа

Awayz (travel API), Lucra Sports (игры на деньги), EatClub (динамические скидки).

Любая программа лояльности становится прибыльной если правильно структурировать экономику.

ПАТТЕРН 2 — ФРАНШИЗА КАК БИЗНЕС-МОДЕЛЬ ДЛЯ ИИ-ПРОФУСЛУГ (новый, Manifest OS $750M)

Manifest OS: 100 независимых юрфирм под единым брендом + общая ИИ-платформа + бэкофис.

Сетевой эффект: больше дел → умнее ИИ → лучше результаты → больше клиентов.

Применимо: медклиники, бухфирмы, архбюро, риелторские агентства, психотерапевты.


ПАТТЕРН 3 — EQUITY ВМЕСТО ПРОДАЖИ (Modus $85M)

Modus: акционер бухфирм → внедряет ИИ изнутри.

Консервативные рынки не покупают SaaS, но берут партнёра-инвестора ради роста прибыли.


ПАТТЕРН 4 — ФИКСИРОВАННАЯ ЦЕНА В ПОЧАСОВЫХ ПРОФУСЛУГАХ

LegalTech ($400M за 286 дней), Manifest OS, M&A брокер, страховой брокер.

Фиксированная цена выравнивает интересы и убирает барьер. Применимо к любым профуслугам с почасовой оплатой.


ПАТТЕРН 5 — ВИДИМОСТЬ КАК ПРОДУКТ

Ontora (процессы компании), Brev (движение к целям), Glimpse (вычеты ритейлеров), Calibre (здоровье в реальном времени).

Компании/люди не видят что происходит → ИИ делает невидимое видимым → это стоит денег.


ПАТТЕРН 6 — ДАННЫЕ КАК ИНФРАСТРУКТУРА

Redpine (Spotify для данных), голосовые данные, база отелей.

99% ценных данных мира закрыты в подписках. ИИ-агенты не могут платить за все подписки → нужен посредник.


ПАТТЕРН 7 — SOCIAL PROOF ДЛЯ ИИ-АГЕНТОВ (новый, Marx)

Агенты учатся у других агентов: Marx (инвестиции), AgentDiscuss (продукты), Humwork (задачи).

Следующий уровень: агенты не просто выполняют задачи — они формируют коллективный интеллект.


ПАТТЕРН 8 — HUMAN-IN-THE-LOOP КАК РОВ

Humwork, Konfide, LegalTech, Imagine AI — ИИ + живой человек как гарант качества.

Работает в высокоставочных решениях где ошибка дорого стоит.


ПАТТЕРН 9 — СКРЫТЫЙ НАЛОГ

ИДЕЯ 2: "Manifest OS для бухфирм и архбюро"

Пересечение: Manifest OS (франшиза юрфирм, $750M) × Modus (equity в бухфирмах)

Manifest OS = франшиза независимых юрфирм под единым брендом + ИИ-платформа.

Тот же паттерн: бухфирмы, архбюро, психотерапевты, диетологи.

Преимущество: сетевой эффект данных — больше клиентов → умнее модель.


ИДЕЯ 3: "Marx для корпоративных решений"

Пересечение: Marx (соцсеть агентов-инвесторов) × B2B SaaS × Ontora (видимость процессов)

Агенты внутри корпорации обсуждают и голосуют по бизнес-решениям.

Аналог: вместо комитетов по закупкам — агенты анализируют вендоров, репутация взвешивает голос.

Применимо: M&A дью дилидженс, выбор поставщиков, стратегическое планирование.


ИДЕЯ 4: "Redpine для медицинских данных"

Пересечение: Redpine (лицензированные данные) × ХелсТех × Amigo (ИИ-агенты для клиник)

99% медицинских данных (история болезней, клинические исследования) закрыты.

ИИ-агентам для диагностики нужны актуальные медданные.

B2B: продавать лицензированный доступ к анонимизированным медданным ИИ-компаниям.


ИДЕЯ 5: "ИИ-аудит комиссий агрегаторов для ресторанов"

Пересечение: Glimpse (вычеты ритейлеров $35M) × EatClub (рестораны) × ИИ-агенты

DoorDash/Instacart берут 15-30% + скрытые вычеты. 1M+ ресторанов США теряют $30-100K/год.

ИИ анализирует выписки → оспаривает необоснованные вычеты. Success fee — zero risk для клиента.


ИДЕЯ 6: "Salesgraph для найма топ-талантов"

Пересечение: Salesgraph (чемпионы внутри компании) × HR-tech / рекрутинг

Лучшие технические кандидаты не реагируют на холодные письма.

Та же логика: найди "чемпиона" внутри компании → снабди его аргументацией для найма конкретного кандидата.

Рекрутинг через внутренних лоббистов.


ИДЕЯ 7: "Франшиза превентивной медицины"

Пересечение: Manifest OS (франшиза юрфирм) × Calibre (превентивная медицина £69/мес)

100 независимых врачей под единым брендом превентивной медицины + общая ИИ-платформа.

Пациент получает качество сети по доступной цене. Врач получает клиентов и платформу.


ИДЕЯ 8: "Пивот-скринер: найди свой HVAC"

Пересечение: Avoca (пивот ресторан→HVAC) × любая ИИ-технология

Инструмент/фреймворк: берёшь любую ИИ-технологию → автоматически ищешь вертикаль с максимальной ценой одного события.

Ресторан $30 → HVAC $30K → медицинская ошибка $300K → авиация $30M.

Применимо к любому ИИ-стартапу на стадии пивота.

КАНАЛ 1 — Y COMBINATOR (~40%). YC = доверие + нетворк + PR. Если целевые клиенты — стартапы и фаундеры, YC самый эффективный B2B-канал.

КАНАЛ 3 — INVITE-ONLY КАК МАРКЕТИНГ. Atlas (luxury карта), Tepali (medspa), Eigen. Эксклюзивность создаёт ажиотаж. Применимо к любому продукту для "элиты" ниши.


КАНАЛ 4 — SUCCESS FEE / NO RISK. LegalTech, M&A брокер, страховой брокер, рекрутинг, Glimpse. Клиент не рискует → низкий барьер входа → рост через результат.


КАНАЛ 5 — ОБРАЗЦЫ ЗА НИЗКУЮ ЦЕНУ. Ковры: образец за $5 + купон $50 на первый заказ. Конверсия через осязаемый опыт для физических товаров с высоким чеком.


КАНАЛ 6 — M&A КАК GTM. Modus и Rowan инвестируют в традиционные компании → получают клиентов + данные + барьер. Не продажи — стратегическая экспансия через владение.

7. ФИЗИЧЕСКИЕ КАСТОМ-ПРОДУКТЫ — ковры подняли $20M без доминирующего конкурента. Тот же паттерн: мебель на заказ, ювелирка, кастомная электроника.

Atlas (invite-only, /год) + Flare (верифицированные контрибуторы):

Пересечение — закрытая сеть верифицированных экспертов с платным доступом.

Например: «Verified Counsel» — юристы/врачи/финансисты с подтверждёнными

Flare верифицирует контент, Eigen создаёт реальные связи — но никто не

верифицирует самих авторов/экспертов в интернете. Кто этот человек реально?

Децентрализованная верификация репутации = незакрытый рынок.

→ Идея: ИИ-агент для field-service вертикалей (стоматология, авторемонт, клининг, рестораны).


2. Trust x Контентный лицензинг (Verification + Copyright)

Flare (верификация интернет-контента) и Sinai (лицензированные ИИ-книги) атакуют: как доверять контенту в эпоху ИИ. Flare — через crowd-верификацию, Sinai — через официальные договоры.

→ Идея: платформа верифицированных профессиональных материалов (медицина, право, финансы) — лицензия + экспертная проверка + ИИ-интерфейс.


3. Luxury x Социальный капитал (Atlas + Eigen)

Atlas (invite-only финкарта) и Eigen (анти-одиночество) монетизируют один ресурс — «доступ к нужным людям».

→ Идея: invite-only профессиональный нетворкинг по модели Atlas ($300-500/год) — ИИ подбирает коллег, консьерж организует встречи. LinkedIn для тех кто ненавидит LinkedIn.


4. UGC x Creator tools x «Roblox для взрослых»

A1Zap = Roblox-механика (UGC как продукт) в взрослой аудитории. Если провалится как соцсеть — выживет как B2B no-code платформа встраиваемых мини-приложений для Telegram/Slack.
